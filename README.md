## � 快速开始

### 1. 克隆项目

```bash
cd python-learning
```

### 2. 运行第一个项目

```bash
cd week1/day1-personal-card
python main.py
```

### 3. 查看代码解析

打开 `week1/day1-personal-card/code_explanation.md` 了解每行代码的含义
