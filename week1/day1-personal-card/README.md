# Day 1: 个人信息卡片生成器

## 🎯 学习目标

- 掌握 Python 基础语法：变量、字符串操作
- 学习 f-string 格式化字符串
- 理解 print 函数的使用
- 体验第一个 Python 程序的成就感

## 📋 项目需求

创建一个程序，能够：

1. 收集用户的个人信息（姓名、年龄、职业、爱好等）
2. 生成格式化的个人名片
3. 支持不同的名片样式
4. 将名片保存到文件

## 🎨 示例输出

```md
=======================================
           个人名片
=======================================
姓名: 张三
年龄: 25岁
职业: 软件工程师
爱好: 编程, 音乐, 读书
邮箱: zhangsan@example.com
座右铭: 永远保持学习的心态
=======================================
生成时间: 2024-01-01 10:30:00
=======================================
```

## 💡 实现思路

1. 使用 `input()` 函数收集用户信息
2. 用变量存储各种信息
3. 使用 f-string 或 `.format()` 格式化输出
4. 创建美观的边框和布局
5. 添加时间戳功能

## 🔧 涉及的语法点

- 变量赋值: `name = "张三"`
- 字符串操作: 连接、重复、切片
- f-string: `f"Hello {name}"`
- print 函数: `print()`
- input 函数: `input("请输入姓名: ")`
- 基本的字符串方法: `.strip()`, `.title()`
